# mse-template

## Установка и запуск

Перед запуском проекта убедитесь, что у вас установлен Docker 
```bash
docker --version
```
```bash
docker compose version
```
Если Docker не установлен, скачайте и установите его с официального сайта:
- Windows / macOS: https://www.docker.com/products/docker-desktop/
- Linux: следуйте инструкции для вашего дистрибутива на https://docs.docker.com/engine/install/


### 1. Клонирование репозитория

```bash
git clone https://github.com/moevm/mse1h2026-review
```
Перейдите в папку проекта
```bash
cd mse1h2026-review
```
#### Альтернативный способ: загрузка архива

Можно просто скачать проект архивом:

1. Перейдите на страницу репозитория:  
   [https://github.com/moevm/mse1h2026-review](https://github.com/moevm/mse1h2026-review)

2. Нажмите кнопку **Code** → **Download ZIP**

3. Распакуйте скачанный архив в удобное место

4. Откройте терминал из распакованной папки проекта

### 2. Настройка окружения
```bash
cp .env.example .env
```
Откройте файл .env в любом текстовом редакторе и укажите ваши данные:
```bash
GITHUB_TOKEN=ghp    - ваш токен доступа (Access Token).
GITHUB_USER=User    - ваш логин на GitHub.
```
### 3. Запуск проекта
```bash
docker compose up -d
```

### 4.  Проверка статуса
```bash
docker compose ps
```
Все сервисы должны быть в статусе Up или healthy.

### 5. Доступ к проекту
После успешного запуска проект будет доступен по адресу: 
```bash
 http://localhost:8080
 ```

 ### 6. Остановка проекта 
```bash
 docker compose down
 ```
 Данная команда останавливает все запущенные контейнеры, но сохраняет данные. При следующем запуске данные не будут потеряны.

 ### 7. Повторный запуск проекта
Для повторного запуска проекта необходимо перейти в папку проекта
```bash
 cd mse1h2026-review
 ```
 и в папке проекта выполнить запуск
 ```bash
docker compose up -d
```
После успешного запуска проект будет доступен по адресу:
```bash
 http://localhost:8080
 ```
